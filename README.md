  - 인풋 애니메이션 기능 구현

  </br>

- 메인 페이지
  - 담당 : 강서윤
  - 캐러셀 슬라이드 구현
  - 신상품 리스트 불러오기 기능 구현

  </br>

  - 페이지 이동 필터 기능 구현

- 상품 전체리스트 페이지

  - 더보기 기능 구현

  - 필터 모달창 통해 검색 기능 구현
   </br>

- 상품 상세페이지
